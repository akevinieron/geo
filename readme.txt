Geo - Aplicacion para geolocalizar lugares en el mapa

Utiliza:

  - API Google Maps JS
  - JSON (transferencia desde-hasta Base de Datos)
  - PHP y MySQL (Manejo en lado de Servidor y Base de Datos)


Caracteristicas:

  - Auto-ubicacion actual (encuentra la posicion actual del usuario en el mapa)
  - Marcar un punto del mapa con el puntero del mouse
  - Ingresar ubicacion manualmente (latitud y longitud exactos)
  - Guardar estas ubicaciones en Base de Datos con la posibilidad de ponerle un nombre, direccion, tipo de lugar
